WannaCry de nos joures

Le ransomware lui-même est moins actif aujourd'hui, mais il reste une menace latente pour les systèmes non protégés et d'autres variantes de ransomwares inspirées par WannaCry continuent de circuler.

  • Le mécanisme de propagation (exploitation d'EternalBlue) a été réutilisé dans d'autres cyberattaques majeures (ex. : NotPetya).

Pour se protéger contre des cyberattaques telles que WannaCry, il est recommandé :

  • D'installer les correctifs logiciels régulièrement, notamment ceux publiés par Microsoft pour combler les vulnérabilités.
  • De sauvegarder régulièrement les données sur des supports non connectés au réseau.
  • De maintenir les outils de protection tels que les logiciels antivirus et antimalware à jour.
  • De former les utilisateurs pour reconnaître les emails de phishing et autres risques de cybermenaces.
  • nstaller tous les correctifs de sécurité disponibles pour votre système d’exploitation et les applications clés
  • Utiliser un système d’exploitation actuel qui inclut les dernières fonctionnalités de sécurité
